### üîç Features

Paper-Workflow includes two main features:

#### 1. HF-DailyPaper

- **Purpose:** Automatically fetches the top three research papers daily from HuggingFace DailyPaper.
- **How it works:** The software generates concise notes for each paper, indicating key elements like authors, summaries, and advantages over baseline papers.
- **Usage:** Set a daily schedule for the workflow, and it will send the notes directly to your chat group via integration with Feishu (Lark).

![HF-DailyPaper](./images/hf-dailypaper.png)

#### 2. ReadingPaper-Lark

- **Purpose:** Analyze specific research papers by entering their PDF URLs.
- **How it works:** The software uses JinaAI Reader to extract and comprehend content from your PDF papers. It generates detailed notes, aiding in understanding the key points and metrics of the study.
- **Usage:** Input the PDF link, and allow the workflow to analyze the paper and send the findings to your Feishu group.
